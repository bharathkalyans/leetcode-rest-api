# leetcode-rest-api
LeetCode API to access the details of a User

Following are the End points
- User
  * user/ (POST)
  * user/contests (POST)
  * user/solved (POST)
  * user/badges (POST)
  * user/activeyears (POST)
  * users/submissions (POST)
- Leetcode
  * leetcode/todaysquestion (POST)
  * leetcode/histogram (POST)
